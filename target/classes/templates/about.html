Acerca de Practica1

Proyecto web convertido a Spring Boot con páginas HTML, recursos estáticos y rutas básicas.

Objetivo

Mantener una plantilla sencilla para practicar navegación, formularios y controladores básicos con Spring Boot.

El proyecto conserva HTML, CSS y JavaScript básico, y agrega un controlador Java para servir las páginas principales.
